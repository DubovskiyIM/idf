# Архив манифестов IDF v1.x

**Актуальный имплементационный статус:** [`../implementation-status.md`](../implementation-status.md)

## Почему архив

До v1.12 манифест вёлся как живой документ с инкрементальными блоками «Изменения в v1.X». В v2 документ переписан как timeless-спецификация о формате, без хроники. Старые версии сохранены как свидетельство эволюции мышления и для обратных ссылок из specs.
