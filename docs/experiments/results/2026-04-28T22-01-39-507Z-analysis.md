## Что это значит для центрального утверждения

До этого run'а у нас было:
> AI-вклад в IDF формате формально неотличим от человеческого (на Opus).

После этого run'а:
> AI-вклад в IDF формате формально неотличим от человеческого **vendor-independently и model-size-independently** в проверенном диапазоне (Opus → R1:8B).

Это **существенно сильнее**:

1. **Format-property, не model-property.** Гипотеза run 4 могла быть «Opus достаточно умный, чтобы пройти our test'ы». Ablation на 8B одной из самых маленьких production моделей **выбивает** этот альтернативный объяснение. Trajectory воспроизводится не из-за Opus, а из-за format'а.

2. **Open-weights baseline.** 4 prior runs стоили $20 на closed-weights API. R1:8B — **$0, локально, open-weights**. Это значит **репликация experiment'а accessible любому исследователю**. Это критично для научной honesty: claim проверяется не только тем, у кого есть Anthropic billing.

3. **Model class universality.** R1:8B — reasoning model (thinking trace). Opus 4.7 — agentic conversational model. Разная архитектура подсказывает — **trajectory не зависит от specific architectural choice**. Open question: standard non-reasoning model той же size (Llama 3.2:3B) даст тот же результат? Это **следующий ablation**.

4. **Cost vs capability decoupled.** На same task'е 8B local дал same outcome quality, что multi-hundred-billion API. Это значит **AI-as-format-citizen не require frontier capability** для этого класса задач. Implication for industry: format-driven AI authoring может работать на edge devices, без vendor lock-in.
